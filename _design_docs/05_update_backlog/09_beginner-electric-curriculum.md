# 09 · 왕초보 일렉 커리큘럼 신설 (1개월 · 손 만들기)

> **이 문서의 성격:** 다른 백로그(01~08)가 *코드/렌더 개선*인 것과 달리, 이건 **신규 커리큘럼 콘텐츠 저작 브리프**다. 세션이 끊겨도 이 문서만 보고 바로 저작에 들어갈 수 있게 모든 설계 결정을 박제한다.
> **다음 세션 시작점:** "아래 §8(16일 골격)·§9(완성물 스케치)를 기준으로 `00_curriculum_authoring_playbook.md` §6 체크리스트를 따라 저작 시작" 하면 됨.

## 목적 (왜 만드는가)

- **비어 있는 핵심 칸:** 기존 3트랙(chord_building·funk_rhythm·solo_scale)은 전부 `level: 2` — "코드는 침 / 곡은 쳐봤음"을 전제한다. PRD의 입문자 페르소나 **'지훈(25)'**("기타 사고 싶은데 독학이 막막함")을 실제로 받아주는 트랙이 **하나도 없다**(갭).
- **유입 최대·착지 없음:** 초보가 기타 인구의 대다수 → 이 트랙이 사이트 최다 유입 커리큘럼이 될 가능성이 높다.
- **입구(gateway) 역할:** `level: 1`로 기존 3트랙 아래에 놓이고, 졸업 후 solo_scale·funk_rhythm·chord_building으로 넘어가는 다리를 놓는다.

## 확정 결정사항

### §1. 포지셔닝
- **level:** `1` (기존 3트랙 아래 입구).
- **durationMonths:** `1`.
- **다음 트랙 연결:** 졸업 시 solo/scale·펑크·코드빌딩으로 안내.

### §2. 성격·범위 (확정)
- **오픈코드 / 스트럼 노래 반주 → 완전 제외** (통기타 노래방 느낌 배제).
- **대상 악기 → 일렉기타 한정.**
- **성격 → 테크닉/메커닉 파운데이션** (크로매틱 · 얼터네이트 피킹 · 파워코드 중심).
- **무게추 → 파워코드** (W3 전체 + W4 대부분). **펜타토닉/솔로는 W4에 맛보기 하루만.**

### §3. 관통 철학
> **"속도가 아니라 깨끗함이 먼저 — 첫 달에 평생 갈 손을 만든다."**
> 목표는 잘 치는 게 아니라 **기타랑 안 헤어지는 것(이탈 방지)**. 매일 끝에 **메트로놈 앞에서 확인되는 결과물 하나**(BPM/리프/릭)를 남긴다.
> — 각 트랙의 "관통 철학"(solo=3도 착지, chord_building=조립)에 대응하는 이 트랙 고유 철학.

### §4. 구조
- **4주 × 4일 = 16일.** (Month→Week→Day, Day = 진도 원자단위)
- **완성물 중첩:** 매일 = 작은 완성 / 매주 = 들려줄 결과물 하나 / 4주 끝 = **졸업 작품(파워코드 곡 녹음)**.

### §5. 하루 루틴 템플릿 (⚠️ 30분 제안 — 최종 미확정)
기존 트랙의 "50분 · 실전 즉흥" 4구간은 왕초보에 부적합(즉흥 불가, 손 피로·이탈). → **30분 · 테크닉 4블록**으로 재정의 제안:

| 블록 | 시간 | 내용 |
|---|---|---|
| ① 워밍업 | 0~7분 | 어제 크로매틱/스파이더 복습(손 풀기 겸) |
| ② 오늘의 기술 | 7~17분 | 새 메커닉, 정확도 우선(느리게) |
| ③ 적용 | 17~27분 | 리프/파워코드에 얹기 (**오늘의 완성물**) |
| ④ 체크 | 27~30분 | 메트로놈 BPM 기록 / 짧게 녹음 |

> **미결정:** 30분(왕초보 이탈·손 피로 고려, 추천) vs 다른 트랙 통일용 50분 유지. **저작 착수 전 확정 필요.** 50분 유지 시 "실전 즉흥" 구간을 "오늘 배운 것 반복 다지기"로 의미 재정의할 것.

## §6. 완성물(악보) 제작 원칙 (확정)

- **전부 오리지널로 Claude가 작곡** → 사이트에 통째 렌더(**저작권 0 · 할루시네이션 0 · 완결**). 짧은 리프 수준이라 자작으로 충분.
- **그럴싸함 = 유명한 음악 규칙 차용:**
  - 마이너 펜타토닉 프레이징(단음 리프)
  - **bVI–bVII–i "록 클라이밍"** (E단조: C5→D5→E5)
  - **i–bVI–bIII–bVII 4코드 루프** (Em–C–G–D — "Zombie"·"With or Without You" 계열)
  - **조용한 벌스(팜뮤트) ↔ 터지는 코러스** 다이내믹 (Nirvana/Pixies 규칙)
- **유명곡은 이름만 "실전 다음 곡"으로 지목**, 탭은 사용자가 외부조달(**우리는 저작권 곡을 호스팅하지 않는다** — 우리 JSON은 오리지널만).
- **오리지널을 목표곡 장르·손동작에 맞춰 작곡** → 치면 "어 이거 그 곡 같네" 소리가 나게.

## §7. 렌더 가능성 확인 (기존 스키마로 전부 커버됨)
- 크로매틱·스파이더 = `type:"tab"` 단음 시퀀스(finger 라벨). ✅
- 파워코드 = `chord[]` 동시타(루트+5도) 또는 2음 스택. ✅
- 팜뮤트 = `technique:"palm_mute"`. ✅ / 데드뮤트 = `dead_note`. ✅
- 해머온/풀오프(W4 맛보기) = `technique:"hammer_on"/"pull_off"`. ✅
- 펜타 박스 = `type:"scale_shape"` 지판 다이어그램(role 색). ✅
- 스윙 필 불필요(정박 록). 슬래시 리듬 미사용(스키마 권고 준수).
- **신규 렌더러 개발 불필요** — 기존 자체 SVG + VexFlow로 전부 커버.

## §8. 16일 골격 (저작 대상)

**W1 — 손 만들기(순수 메커닉)** · 주간 완성물: **스파이더 크로매틱 6→1줄 클린(60~70 BPM)**
- **D1** 자세·피크 잡기 + 다운피킹, 한 줄 균일 타격 → 완성: 6번줄 8분음표 다운피킹 균일(60~70)
- **D2** 왼손 1지1프렛 + 크로매틱 1-2-3-4 → 완성: 버징 없이 6번줄 1-2-3-4
- **D3** 얼터네이트 피킹(다운-업) 한 줄 → 완성: 얼터네이트 8분음표 균일
- **D4** 크로매틱+얼터네이트+줄 이동 → 완성: 스파이더 6→1→6 클린(60~70)

**W2 — 두 손 싱크 + 첫 리프** · 주간 완성물: **얼터네이트/팜뮤트 단음 리프**
- **D1** 메트로놈에 두 손 싱크 + 손가락 독립(1-3-2-4 변형)
- **D2** 줄 넘는 피킹(inside/outside) → 2~3줄 걸친 패턴
- **D3** 팜뮤트 도입 + 다운피킹 갈기(chug) → 완성: 6번줄 팜뮤트 그루브
- **D4** 유명 단음 리프급 오리지널을 목표 BPM까지 (실전 다음 곡: Seven Nation Army / Come As You Are)

**W3 — 파워코드 입문** · 주간 완성물: **팜뮤트 파워코드 리프**
- **D1** 파워코드 폼(루트+5도, 6·5번줄 루트) + 안 쓰는 줄 뮤트 → 완성: 잡음 없는 파워코드 한 개
- **D2** 파워코드 이동 + 프렛↔음이름 감 → 완성: 두 파워코드 클린 왕복
- **D3** 파워코드 + 팜뮤트 + 다운피킹 리듬 → 완성: 팜뮤트 파워코드 리프(정박)
- **D4** 파워코드 3~4개 진행 리프 녹음 (실전 다음 곡: You Really Got Me / Iron Man)

**W4 — 파워코드 곡 완성 + 솔로 맛 + 다음 문** · 주간 완성물: **파워코드 록 곡(졸업 작품)**
- **D1** 파워코드 리듬 확장(쉼표·싱코페이션·옥타브) → 리프에 표정
- **D2** 리프들을 파트로 잇기(벌스/코러스 느낌) + 멈추지 않고 넘어가기
- **D3** 〈맛보기〉 마이너 펜타토닉 박스1 + 해머온/풀오프 살짝 → 첫 3음 릭
- **D4** **졸업:** 파워코드 곡 풀 녹음 + 4주 BPM 성장표 + 다음 트랙(solo/scale·funk·chord_building)으로 다리
  (실전 다음 곡: Smells Like Teen Spirit / Song 2)

## §9. 완성물 스케치 (개념안 — 다음 세션에 JSON화, 탭은 개념용)

**W1 · 스파이더 크로매틱** (곡 아님, 순수 연습)
```
E|-1-2-3-4-----------------...--|  각 줄 1-2-3-4 상행
A|---------1-2-3-4---------...--|  6→1줄 후 4-3-2-1 하행
```
완성 = 60~70 BPM 클린.

**W2 · 단음 리프** — E단조 펜타, 팜뮤트, 얼터네이트
*빌린 규칙: 반복 루트 + 상행 도약 후 착지(펜타 프레이징)*
```
E|-0-0-0-3-5-3-0---0-0-0-3-0---|
   E E E G A G E    E E E G E     ← 5(A)에 악센트, E로 착지
```

**W3 · 파워코드 리프** — 팜뮤트 chug + 클라이밍 훅
*빌린 규칙: bVI–bVII–i Aeolian 클라이밍*
```
D|-9-9-9-9---5-7-9-|
A|-7-7-7-7---3-5-7-|   ← 같은 파워코드 폼을 3→5→7 슬라이드
   E5 팜뮤트×4  C5 D5 E5
```

**W4 · 파워코드 곡(졸업)** — 벌스+코러스
*빌린 규칙: i–bVI–bIII–bVII 4코드 루프 + 조용-시끄러움 다이내믹*
```
[벌스] 팜뮤트 E5 펄스 + W3의 C-D-E 클라이밍을 필(fill)로
[코러스] 울리는 다운스트로크: E5 ── C5 ── G5 ── D5  (Em–C–G–D 루프)
   E|0    ─    3    ─
   A|2    3    5    5
   D|─    5    ─    7
```
