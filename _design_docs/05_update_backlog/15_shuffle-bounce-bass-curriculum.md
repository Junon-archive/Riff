# 15 · 셔플 & 바운스 베이스 커리큘럼 (2개월 · 필 트랙)

> **성격:** 신규 커리큘럼 저작 브리프. **구 "그루브 & 포켓(1개월)"에서 바운스/셔플 필 트랙으로 전면 개편(2026-07-10)** — 개념이 추상적이던 걸 구체·장르 정체성으로 교체하고 2개월로 확장. 저작 SOP·프롬프트법 = [[09-beginner-electric-curriculum]] §12·§13.

## 목적·포지셔닝
- **채우는 칸:** 베이스 **'필(feel)/타이밍'** — 루트는 짚지만 그루브가 안 튀는 벽을 넘긴다. 필을 **스트레이트 → 스윙8(셔플) → 스윙16(바운스)** 로 진화.
- **id:** `shuffle_bounce_bass_2months` · **level:** 2 · **instrument:** bass · **durationMonths:** 2 · **estMinutes:** 50.
- **선수:** 베이스 기초([[13-beginner-bass-curriculum]]).

## 관통 철학
> **"박자는 격자가 아니라 삼각형 — 8분음표를 롱-숏으로 굴리면 그루브가 튄다."**
> 포켓의 목표를 '정확한 타이밍'에서 **'굴러가는 타이밍(바운스)'**으로 옮긴다. 바운스의 정체 = **트리플렛 그리드 + 고스트노트 + 레이백**.

## 차별화 (겹침 방지)
- **워킹(16)** = 코드 위 라인 '작곡'(하모니) / 이 트랙 = 같은 음을 **'어떻게 굴리나'(필)**.
- **슬랩(14)** = 손 기술 / 이 트랙 = 스윙 필.
- **블루스(11, 기타)** = 장르 전반 / 이 트랙 = 그 셔플을 **베이스 필로**. **M1 셔플이 블루스(11) 트랙과 키·진행이 맞아 크로스 합주 가능.**

## ★ 4현·5현 병행 제시 (모든 베이스 트랙 공통)
- 규약 상세 = [[13-beginner-bass-curriculum]] "4현·5현 병행 제시 규약". 같은 그루브를 `stringCount:4`·`5` 두 블록으로.
- **5현 활용:** 저음 B로 더 묵직한 부기/바운스 루트 옵션(+B현 뮤팅 과제). 부기 워크를 5현 저역으로 확장하는 대안 표기는 5현 블록에만.

## 렌더·의존 (엔진 준비 완료)
- ✅ **엔진(10) B0~B4 완료** — 4·5현·베이스 클레프. 신규 렌더러 불필요.
- **스윙 표기:** `meta.feel:"swing8"`(셔플)·`"swing16"`(바운스) — 악보는 정박 표기 + 스윙 지시(산문).
- **트리플렛 명시:** 백로그 02-C(튜플렛) 완료 → **학습 초반 "롱-숏" 그리드 시각화**에 셋잇단음표를 직접 그림.
- **고스트/뮤팅:** `dead_note`(음정 없는 "칙")·`rest`(공간). 
- **표기 전략:** 초반=트리플렛 명시(그리드 보이기) → 그루브=`swing8/16` 단축표기 **병용**.

## 완성물 중첩
- **매일** = 작은 필 완성물 / **매주** = 주간 그루브 / **월** = M1 12마디 셔플 · M2 바운스 졸업 그루브.

## 범위·구성 (2개월 = 8주 32일)

### Month 1 — 셔플 (swing8) · 월 완성물 → *12마디 블루스 셔플 부기 완주*
**W1 — 삼각형 느끼기(트리플렛→스윙)** · 완성 → 셔플 루트 펄스
- D1 셋잇단음표 그리드("1-트-플-릿"), 트리플렛 명시로 시각화 · D2 스트레이트→스윙, 롱-숏(가운데 음 빼기) · D3 셔플 루트 펄스(swing8)를 메트로놈에 · D4 2코드 위 셔플 루트

**W2 — 셔플 루트 & 부기 워크** · 완성 → 부기 셔플 그루브(1~2코드)
- D1 루트 + 스윙 뒷음("a")에 얹기 · D2 부기 워크 상행 **R-5-6-b7** · D3 부기 왕복(상·하행) · D4 부기 셔플 그루브

**W3 — 12마디 셔플(I-IV-V)** · 완성 → 12마디 셔플 라인
- D1 12마디 폼 + I 부기 · D2 IV·V로 부기 폼 이동 · D3 12마디 전체 순회(전환 매끄럽게) · D4 12마디 셔플 라인

**W4 — 셔플 굳히기·변주·녹음(M1 졸업)** · 완성 → **12마디 블루스 셔플 완주**
- D1 셔플 텍스처(고스트 살짝·다이내믹) · D2 턴어라운드/엔딩 · D3 리허설(멈추지 않고 12마디) · D4 **M1 녹음** 〈블루스 기타와 합주 가능〉

### Month 2 — 바운스 (swing16 + 고스트 + 레이백) · 월 완성물 → *바운스 졸업 그루브 녹음*
**W5 — 고스트노트 바운스** · 완성 → 고스트 튀는 그루브
- D1 데드노트 "칙"(음정 없는 뮤트) · D2 루트 + 고스트 교대 · D3 뮤팅으로 공간 만들기 · D4 고스트 튀는 그루브

**W6 — 스윙16 바운스(네오소울/힙합)** · 완성 → 스윙16 그루브
- D1 16분 그리드 + 스윙16 감(뒤 16분 미룸) · D2 네오소울 바운스 루트 라인 · D3 고스트+스윙16 결합 · D4 스윙16 그루브(1~2코드)

**W7 — 레이백 & 포켓 + 〈맛보기〉하프타임 셔플** · 완성 → 레이백 그루브 + 하프타임 맛
- D1 뒷박에 눕기(behind the beat) 감각 · D2 앞/정박/뒤 밀당 대비(같은 라인 3위치) · D3 **〈맛보기〉하프타임 셔플 필**(스네어 2·4·스윙16) · D4 레이백 그루브

**W8 — 졸업 · 종합 바운스 그루브** · 완성 → **바운스 졸업 그루브 녹음**
- D1 스윙16+고스트+레이백 종합 · D2 곡 구조(벌스/코러스 그루브 변주) · D3 리허설(멈추지 않고) · D4 **졸업 녹음** + 8주 성장(스트레이트→셔플→바운스) 회고 + 다음 트랙 다리(슬랩·워킹)

## 완성물 스케치 (다음 세션 4/5현 병행 JSON으로 승격)

**M1 셔플 부기(핀 고정 후보)** — E 부기, `meta.feel:"swing8"`. 부기 워크 **R-5-6-b7**:
- E 루트=4번현 0 · 5도 B=3번현 2 · 6도 C#=3번현 4 · b7 D=3번현 5 (상행 후 6-5로 하행). 셔플 필(롱-숏)로 8분. → 아이코닉 블루스 셔플 부기. 12마디는 I(E)-IV(A)-V(B)로 같은 폼 이동.
- **5현:** 운지 동일 + 저음 B현으로 더 묵직한 루트 대안.

**M2 바운스(졸업 후보)** — `meta.feel:"swing16"`. 루트 + 고스트(dead_note) + 레이백:
- 1박 루트, 뒤 16분에 고스트("칙"), 당김음으로 뒷박 강조. 네오소울/힙합 바운스 느낌. (정확한 마디는 JSON 승격 시 확정.)

- 전량 오리지널(저작권 0), 유명 규칙 차용(셔플 부기 R-5-6-b7·12마디 I-IV-V·스윙16 바운스). 유명 셔플/바운스 곡은 이름만 지목. 주간·월 완성물을 프롬프트에 **핀 고정**([[09-beginner-electric-curriculum]] §6).

## 데일리 50분 루틴 (③ 오늘의 연습)
- **0~10 워밍업:** 트리플렛/스윙 그리드를 손+입으로 세기 + 어제 그루브 손 풀기
- **10~20 두뇌 훈련:** 오늘 필을 메트로놈/드럼에 느리게(롱-숏·고스트·레이백을 정확히)
- **20~40 실전 그루브:** 오늘 그루브를 드럼/백킹에 얹어 반복(**오늘의 완성물**, BPM 구체 지정)
- **40~50 녹음/셀프 피드백:** 녹음 후 체크 — "그루브가 튀는가 / 뒷박에 눕는가 / 고스트가 사는가 / 드럼 킥·스네어와 맞물리는가"
