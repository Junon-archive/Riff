# 14 · 슬랩/펑크 베이스 커리큘럼 (2개월)

> **성격:** 신규 커리큘럼 콘텐츠 저작 브리프. 저작 SOP·프롬프트 작성법은 [[09-beginner-electric-curriculum]] §12·§13, 4/5현 병행 규약은 [[13-beginner-bass-curriculum]] 재사용.

## 목적·포지셔닝
- **채우는 칸:** 베이스에서 가장 "배우고 싶다" 수요가 몰리는 **후킹 트랙**(슬랩). 기타 `funk_rhythm_2months`와 짝.
- **id:** `slap_funk_bass_2months` · **level:** 2 · **instrument:** bass · **durationMonths:** 2 · **estMinutes:** 50.
- **순서(확정):** 입문(13) 다음의 **병렬 브랜치** — 슬랩(14)·바운스(15) **둘 다 level 2**로, 학습자가 골라 들어온다. 슬랩은 최대 동기부여 카드라 입문 바로 다음 선택지로 노출.
- **self-contained:** 선수는 **베이스 기초(13)만**. 15(필/타이밍)를 먼저 하면 도움되지만 필수 아님 → **슬랩용 타이밍·뮤팅을 이 트랙 안에서 처음부터** 가르친다.

## 관통 철학
> **"슬랩은 손가락이 아니라 드럼 — 엄지로 치고(킥), 손가락으로 튕기고(스네어), 고스트로 그 사이를 채운다."**
> 매일 끝에 결과물 하나. **W1 안에 썸+팝을 둘 다 넣어 첫 주에 '슬랩 소리'가 나게** 한다(리텐션 우선 — 일주일 내내 썸만 치면 이탈).

## 차별화 (15 셔플&바운스와 겹침 방지)
공유 표면(16비트·고스트·싱코페이션·펑크 느낌)이 있으나 축이 다르다:

| | **14 슬랩/펑크** | **15 셔플 & 바운스** |
|---|---|---|
| 축 | **테크닉(슬랩 손) + 펑크 어휘** | **필/타이밍(핑거스타일)** |
| 핵심 질문 | "어떻게 그 **퍼커시브 슬랩 소리**를 내나" | "음을 **언제** 놓나(스윙/레이백)" |
| 주법 | 썸/팝/좌수슬랩 (**신규 손기술**) | 투핑거 핑거스타일 (입문 연장) |
| 리듬 | **스트레이트 16비트** 펑크 | **스윙8(셔플)·스윙16(바운스)** |
| 고스트노트 | 슬랩 사이 **퍼커션 텍스처**(좌수/데드) | 스윙·포켓의 **타이밍 장치** |

**경계 규칙 한 줄:** *15 = 스윙된 필(핑거스타일) / 14 = 스트레이트 16비트 슬랩(손기술).* 고스트가 둘 다 나오지만 14는 **슬랩으로 만든 스트레이트 텍스처**, 15는 **핑거로 굴리는 타이밍**이라 손·목적이 다르다. 둘을 다 들으면 "필 + 슬랩"으로 합쳐진다(상호보완).

## ★ 4현·5현 병행 제시 (모든 베이스 트랙 공통)
- 규약 상세 = [[13-beginner-bass-curriculum]] "4현·5현 병행 제시 규약". 같은 그루브를 `stringCount:4`(EADG) + `5`(BEADG) 두 블록.
- **슬랩 고유 5현 활용:** 13은 저음 B를 주로 "뮤팅"만 했지만, **슬랩에선 저음 B가 실제로 유용** — 더 묵직한 썸 그루브, 저역 확장. 옥타브 그루브를 B현 루트로 내리는 대안을 5현 블록에서 적극 소개(4/5 설명이 풍부해지는 지점).

## 완성물 중첩 (매일 / 매주 / 월)
- **매일** = 작은 완성물 / **매주** = 주간 그루브 / **월** = M1 옥타브 슬랩 그루브 · M2 펑크 슬랩 그루브 곡(졸업).

## 범위·구성 (2개월 = 8주 32일)

### Month 1 — 슬랩 테크닉 · 월 완성물 → *다듬은 옥타브 슬랩 그루브(고스트 포함) 녹음*
**W1 — 썸 & 팝, 둘 다 소리 내보기** · 완성 → 느린 썸-팝 교대(내 첫 슬랩 소리)
- D1 썸 동작·톤(팔뚝 회전·튕겨 나오기, 개방현) · D2 썸 줄이동·뮤팅(E·A) · D3 **팝 동작**(고음현 걸어 튕기기) · D4 **썸-팝 첫 만남(T-P 교대)**

**W2 — 썸·팝 다루기(코디네이션 + 옥타브 + 첫 그루브)** · 완성 → 러프한 옥타브 슬랩 그루브
- D1 T-P 균일·타이밍(메트로놈) · D2 **옥타브 모양**(썸=루트, 팝=옥타브) · D3 T-P 리듬 패턴(8/16 배치) · D4 첫 슬랩 그루브(옥타브 초안)

**W3 — 고스트 & 뮤팅(바운스 엔진)** · 완성 → 고스트가 튀는 그루브
- D1 좌수슬랩/데드 "칙" · D2 T-고스트-P · D3 양손 뮤팅(안 친 줄 침묵) · D4 고스트 그루브

**W4 — 옥타브 슬랩 그루브 완성(M1 졸업)** · 완성 → 다듬은 옥타브+고스트 그루브 녹음
- D1 그루브 다듬기 · D2 다이내믹/변주 · D3 리허설(멈추지 않고) · D4 녹음 〈M1 결과물〉
- 진화 라인: **W2 러프 → W3 고스트 얹기 → W4 다듬어 녹음**(옥타브 그루브가 3주에 걸쳐 펑키해짐, 중복 없음).

### Month 2 — 펑크 응용 · 월 완성물 → *클래식 펑크 슬랩 그루브 곡 녹음(졸업)*
**W5 — 16비트 싱코페이션** · 완성 → 싱코페이션 슬랩 그루브
- 스트레이트 16 그리드에 T/P/고스트 배치, 당김음.

**W6 — 해머온/풀오프 + 좌수 라인** · 완성 → 슬랩+레가토 릭
- 슬랩에 프렛손 해머/풀 얹기, 펜타토닉 슬랩 릭.

**W7 — 팝핑 멜로디 + 더블스탑 (+ 〈맛보기〉 더블 썸)** · 완성 → 팝핑 멜로디 라인
- 옥타브·두 음 동시(더블스탑) 팝핑. **더블 썸(down-up 썸)은 하루 맛보기로만**(고난도 → "다음 단계" 포인터).

**W8 — 졸업 · 클래식 펑크 슬랩 그루브 곡** · 완성 → 펑크 슬랩 그루브 녹음
- 배운 요소 종합(옥타브·고스트·싱코페이션·팝핑) → 한 곡. **클래식 펑크 지향(Larry Graham·Flea 계열 느낌, 이름만 지목).** 4주 성장 회고 + 다음 트랙(바운스·워킹) 다리.

## 완성물 스케치 (다음 세션 4/5현 병행 JSON으로 승격)

**M1 옥타브 슬랩 그루브(핀 고정 후보)** — E 옥타브 그루브, 8분 그리드 **T-고스트-P-고스트**:
- 썸 루트 E = 4번현 0 (`technique:"slap_thumb"`), 팝 옥타브 E = 2번현 2 (`technique:"slap_pop"`), 사이 고스트 = `dead_note`.
- 패턴(8분): T(1) · ghost(1&) · P(2) · ghost(2&) · T(3) · ghost(3&) · P(4) · ghost(4&).
- **5현:** 운지 동일 + **저음 B현 루트 대안**으로 더 묵직한 버전 소개.

**M2 펑크 슬랩 그루브(졸업 후보)** — 스트레이트 16, 옥타브+고스트+싱코페이션+팝핑 릭 결합. 클래식 펑크 색. (정확한 마디는 JSON 승격 시 확정.)

- 전량 오리지널(저작권 0), 유명 규칙 차용(옥타브 슬랩·펜타 슬랩 릭·16비트 펑크). 유명 슬랩 라인은 이름만. 주간·월 완성물을 프롬프트에 **핀 고정**.

## 데일리 50분 루틴 (③ 오늘의 연습)
- **0~10 워밍업:** T-P 교대·크로매틱 슬랩 손 풀기 + 어제 그루브.
- **10~20 두뇌 훈련:** 오늘 기술 느리게(썸 타점·팝 위치·고스트 정확히).
- **20~40 실전 그루브:** 드럼/백킹에 오늘 그루브 반복(**오늘의 완성물**, BPM 지정).
- **40~50 녹음/셀프 피드백:** 썸 톤 균일한가 / 팝이 아프지 않고 또렷한가 / 고스트가 사는가 / 안 친 줄이 조용한가.
