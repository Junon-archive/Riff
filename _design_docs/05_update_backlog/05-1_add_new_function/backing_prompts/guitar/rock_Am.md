## 코드 진행 (in A minor)

```
| Am | F | C | G |   (× 4회 = 16마디)
```

- i–♭VI–♭III–♭VII. 전 세계 락/팝의 표준 루프이자 **A 마이너 펜타토닉이 처음부터 끝까지 그대로 맞는** 진행 → 즉흥 입문에 최적.
- 코드톤 타겟팅 연습용 3도: Am→C · F→A · C→E · G→B.

## 비고 / 루프 트림 메모

- 4마디 진행이 **네 번 똑같이** 반복되는 구간을 고른다. Suno가 후반에 코드를 바꾸거나 빌드업(드럼 추가/기타 레이어 증가)을 넣는 경우가 많으니 **다이내믹이 평평한 16마디**를 선별.
- 트림 기준점은 **1마디 1박 Am의 어택**. 16마디 G 끝 → 1마디 Am으로 되돌아갈 때 크래시 심벌 잔향이 걸리면 짧은 크로스페이드.
- 리드 기타/신스 멜로디가 얹힌 테이크는 폐기(사용자 솔로와 충돌).
