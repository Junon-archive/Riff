# Week 3 — ドラムと一体に：キックにルートを乗せる
> カリキュラム: beginner_bass_1month · Month 1 · 上位: ../month_1_overview.md · dayKey: m1.w3

ベースは一人で目立つ楽器ではありません。<mark>キックドラムと一体</mark>になるとき、曲が硬く締まります。今週は8分音符のルートをキックに正確に乗せ、間の<mark>休符（空間）</mark>でグルーヴを作ります。サブディビジョンを体で数え、二つのコードの上でルートを揺れずに移し、ドラムとぴったりかみ合う感覚を手に入れます。

## 今週の目標
8分音符を体で数えてキックにルートを乗せ、一つのコードの上で8ビートのルートラインを均一に弾き、二つのコードの転換をドラムの上で滑らかにし、最終日にはE-A 8ビートのルートライン（ポケットあり）をメトロノーム75で揺れずに弾くこと。

## 今週の道のり
- **Day 1** — サブディビジョンを感じる（8分を数える）＋ キックにルートを乗せる。
- **Day 2** — 8分音符のルートライン（一コード）＋ 中間の休符でポケットを作る。
- **Day 3** — 2コードの転換をドラムの上で。コードが変わるときルートを追う。
- **Day 4** — 8ビートのルートライン曲（2コード）を完成。

## コアの感覚
- **一人で目立たない。** キックと同じ瞬間に手が落ちる。
- **休符がグルーヴを作る。** 手は止めず、その場所は空ける。
- **5弦はB弦を寝かせ続ける。** グルーヴの中でもミュートを保つ。
