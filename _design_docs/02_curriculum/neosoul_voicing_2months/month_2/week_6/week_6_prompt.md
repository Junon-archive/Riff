# 표준 JSON 스키마 · 필드 규약
- id: `"m2.w6.d{일}.슬러그"`. type: `fretboard_diagram` | `tab`. meta: `{ title, stringCount:6, tuning:["E","A","D","G","B","E"], key?, tempoBpm?, notation? }`.
- 현 번호 6(저음E)~1(고음e). 도수 label에만. ⛔ 스키마 밖 필드명 금지. **풀오프=`pull_off`·해머=`hammer_on`·슬라이드=`slide`.** ※ 그레이스노트 전용 필드는 없으니 **짧은 앞음+해머/풀로 근사**(산문 설명). `notation:"staff+tab"`. ⛔ `"rhythm"`. role root/chord_tone→파랑, target→초록. ⛔ 빨강 없음.
