# Week 3 — 6・5弦ルートを混ぜてII-V-Iを最短接続（ボイスリーディング）
> カリキュラム: chord_building_2months · Month 1 · 上位: ../month_1_overview.md · dayKey: m1.w3

1週目は6弦ルート、2週目は5弦ルートを別々に覚えました。今週はその2つを**混ぜて使います。** ポピュラー音楽の心臓である**II-V-I**（例：CキーのDm7 → G7 → Cmaj7）を弾くとき、手をネック上で大きくジャンプさせず、**ルート弦を6弦・5弦から選んで**3つのコードをほぼその場でつなぎます。これがプロの言う**ボイスリーディング** — コードが変わるとき音が最短距離で滑る技術です。

## 今週の目標
CキーのII-V-Iを**Dm7（5弦）→ G7（6弦）→ Cmaj7（5弦）**で、手の移動を最小化して滑らかに循環すること。各コードのルートが6弦か5弦かを毎回認識し、ガイドトーン（3度・7度）が次のコードへ半音ずつしか動かないのを目と耳で確認するのがゴールです。

## 今週の旅
- **Day 1** — II-V-Iとは何か、なぜポップスの心臓か。「なぜ遠くへ行かずにコードが変わるのか」＝ルート弦交差の概念。
- **Day 2** — CキーでDm7（5弦）-G7（6弦）-Cmaj7（5弦）を同じフレット区域で接続。3コードのルート弦の違い＋指の最小移動。
- **Day 3** — 同じパターンを別キーで（DキーのEm7-A7-Dmaj7）。II-V-Iは丸ごと動く「移動可能なパターン」だと体得。
- **Day 4** — 週の統合。バッキング上でII-V-Iを手の移動最小で滑らかに循環。切替時に「ジャンプしない」感覚を完成。

## コアの感覚
- **ルート弦を選んで手を1区域に縛る。** 6弦コードと5弦コードを交差すればネックを走り回らなくていい。
- **ガイドトーン（3度・7度）は半音ずつしか動かない。** コードが変わっても共通音はそのまま、残りは少しだけ。
- アマチュアはコードを「かたまり」で変え、プロは**最も近い音**へつなぐ。
