# Week 3 — 6·5현 근음을 섞어 II-V-I 최단 연결 (보이스 리딩)
> 커리큘럼: chord_building_2months · Month 1 · 상위: ../month_1_overview.md · dayKey: m1.w3

1주차엔 6현 근음, 2주차엔 5현 근음을 따로 익혔죠. 이번 주는 그 둘을 **섞어 씁니다.** 대중음악의 심장인 **II-V-I**(예: C키의 Dm7 → G7 → Cmaj7)을 연주할 때, 손을 넥 위에서 크게 점프시키지 않고 **근음 줄을 6번·5번으로 골라** 세 코드를 거의 제자리에서 잇는 거예요. 이게 프로들이 말하는 **보이스 리딩** — 코드가 바뀔 때 음들이 최소 거리로 미끄러지는 기술이에요.

## 이번 주 목표
C키 II-V-I을 **Dm7(5현) → G7(6현) → Cmaj7(5현)** 로, 손 이동을 최소화해 부드럽게 순환하는 것. 각 코드의 근음이 6번인지 5번인지 매번 인지하고, 가이드톤(3도·7도)이 다음 코드로 반음씩만 움직이는 걸 눈과 귀로 확인하는 게 결승선이에요.

## 이번 주 여정
- **Day 1** — II-V-I이 뭔지, 왜 대중음악의 심장인지. "왜 멀리 안 가고도 코드가 바뀌는가" = 근음 줄 교차 개념 소개.
- **Day 2** — C키에서 Dm7(5현)-G7(6현)-Cmaj7(5현)을 같은 프렛 구역에서 연결. 세 코드 근음 줄 차이 + 손가락 최소 이동.
- **Day 3** — 같은 패턴을 다른 키로(D키: Em7-A7-Dmaj7). II-V-I은 통째로 움직이는 '이동 가능한 패턴'임을 체득.
- **Day 4** — 주간 통합. 백킹 위에서 II-V-I을 손 이동 최소화로 부드럽게 순환. 전환 시 '점프하지 않는' 감각 완성.

## 핵심 감각
- **근음 줄을 골라 손을 한 구역에 묶는다.** 6현 코드와 5현 코드를 교차하면 넥을 안 뛰어다녀도 돼요.
- **가이드톤(3도·7도)은 반음씩만 움직인다.** 코드가 바뀌어도 공통음은 그대로, 나머지는 살짝만.
- 아마추어는 코드를 '덩어리'로 바꾸고, 프로는 **가장 가까운 음으로** 잇는다.
