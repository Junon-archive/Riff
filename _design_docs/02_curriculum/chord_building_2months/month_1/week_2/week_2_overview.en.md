# Week 2 — Moving down to the 5th-string root (A-form): Maj7 arrives
> Curriculum: chord_building_2months · Month 1 · Parent: ../month_1_overview.md · dayKey: m1.w2

Last week the root lived on the **6th string.** This week we move that home to the **5th string.** Only the root string changed — the rule ("stack the 3rd and 7th over the root and switch with a finger") is identical. But the layout shifts, so we re-learn where R·3·5·7 landed. And this week's new weapon is **Maj7** — a chord exactly a half step from the dominant 7 (b7). With it, you burn the subtle-but-decisive color difference of **7 vs Maj7** into your ear.

## This week's goal
Over a 5th-string root, fret R·3·5 and switch all five characters (Maj·m·7·m7·Maj7) by finger alone: **2nd string = the 3rd switch (3↔b3),** **3rd string = the 7th switch (R→7→b7).** The finish line is a solid 6th-string mute habit and hearing the half-step difference between 7(b7) and Maj7(7).

## This week's journey
- **Day 1** — Dissect the 5th-string-root major (A-form). The feel of the root dropping to the 5th string, muting the 6th, the new R·3·5 layout.
- **Day 2** — Major → minor (2nd string 3↔b3) + dominant 7 (3rd string 5→3). Week 1's principle, unchanged on the 5th-string root.
- **Day 3** — Build m7, then **Maj7 arrives.** On the 3rd string, put b7 (fret 3) next to 7 (fret 4) — the color a half step makes.
- **Day 4** — Weekly integration. Cycle all five characters (Maj/m/7/m7/Maj7) over one 5th-string root.

## Core feel
- **The root is the 5th string. Kill the 6th string firmly.**
- **The 3rd string is the 7th's three-way** — fret 5 (R), fret 4 (Maj7), fret 3 (b7). Three branches on one string.
- Hear, by ear, the half-step difference between 7 (dominant, tense) and Maj7 (a soft unfinished — that jazz color).
