# Month 2 — 고음역·색채·탑노트: 컴핑을 예술로
> 커리큘럼: chord_building_2months · Month 2

1개월차에 코드를 '조립'하는 법(근음 위 3도·7도 인터벌 스위치)을 배웠죠. 이번 달은 그 위에 **색과 노래**를 입힙니다. 근음을 4번 줄로 올려 **고음역**을 열고, 9도 **텐션**으로 색채를 더하고, 코드가 바뀔 때 **탑노트가 멜로디처럼** 이어지게 만들고, 마지막엔 배운 걸 전부 섞어 **나만의 컴핑을 녹음**합니다.

## 이번 달 여정
- **Week 5** — 4현 근음(D 폼) 고음역 트라이어드. '탑노트' 개념의 시작.
- **Week 6** — 색채 더하기. m7·Maj7에 새끼손가락으로 **9도(9th) 텐션**을 얹기.
- **Week 7** — 실전 보이스 리딩. 코드가 바뀔 때 **탑노트 멜로디 라인**을 부드럽게 잇기.
- **Week 8** — 코드 메이커 레코딩 챌린지. 배운 모든 보이싱을 섞어 창의적으로 반주하고 녹음·분석.

## 끝나면 달라지는 것
코드를 '잡는' 사람에서, **탑노트로 노래하고 텐션으로 색을 입히는** 사람이 됩니다. 백킹 위에서 자기만의 컴핑을 설계할 수 있어요.
