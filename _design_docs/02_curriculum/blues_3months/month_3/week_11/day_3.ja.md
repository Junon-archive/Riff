# Day 3 — 拍の後ろに少し寝る余裕

## ① 理論・説明

スライドも、ベンディングも、ビブラートも、もう手に馴染みました。今日は道具ではなく**余裕**を学びます — それが**レイバック**です。レイバックは拍にぴったり合わせる代わりに、ほんの少し後ろに寝るように遅れて弾くフレージングです。<mark>拍の後ろに少し寝ると想像してみてください — 焦らず一拍の余裕を置けば、リックに大人のゆったりさが生まれます。</mark>焦って前に出る代わりに、半拍後ろで気楽に歌うのです。

レイバックは拍を「間違える」ことではありません。バッキングの拍は正確に流れ、皆さんのリードだけが**少し後ろから**ついていくのです。だから<mark>心の中では拍をはっきり数えていてこそ、その上で余裕をもって遅らせられます。</mark>今日はルートAを長く**ビブラート**で伸ばしながら、焦らず後ろに寝る感覚を覚えます。**長い音**を恐れず、その余白がリックをより大きく聞かせることを感じてみてください。

レイバックと対をなすのが**ダイナミクス**です。すべての音を同じ強さで弾くと平坦ですが、ある音は弱く、ある音は強く弾くとリックが生きます。<mark>大事な音は大きく、通り過ぎる音は小さく — この強弱の波がフレージングに表情を加えます。</mark>今日はルートの着地を特に強く余裕をもって、残りはやわらかく流します。**BPM 70**で、後ろに寝る余裕と強弱を一緒に感じながら練習します。速く弾くより余裕をもって弾くのが今日の本当の目標です。一つの音の余白がどれだけ大きく聞こえるか、自分で感じてみてください。急ぐときこそ、むしろゆっくり進みます。

**10〜20分・頭のトレーニング（今日のターゲット＝レイバック）**
実戦の前に、バッキングの拍を心の中ではっきり数えながら、リードを少し後ろに寝かせる感覚を想像します。<mark>拍を正確に知っていてこそ、その上で余裕をもって遅らせられます — レイバックは拍を「知っている人」の余裕です。</mark>

**20〜40分・実戦：余裕のあるフレージングリック（BPM 70）**
今日の完成物、**レイバックリック**です。ルートAを長く**ビブラート**で伸ばして余裕を作り、b3・5度をやわらかく流します。急いで埋めず、余白と強弱でリックを歌います。

## ④ ヒント / よくあるミス

レイバックでよくあるミスです。ほとんどは**余裕に耐えられず焦る**ところから来ます。

```json

  "meta": { "title": "Don't fear the space", "stringCount": 6, "tuning": ["E","A","D","G","B","E"], "key": "A minor" },
  "fretboard": { "startFret": 5, "fretSpan": 4, "dots": [
    { "string": 4, "fret": 7, "finger": 3, "label": "R", "role": "target", "highlight": true },
    { "string": 3, "fret": 5, "finger": 1, "label": "b3", "role": "scale" }
  ]}
}
```
▶ **余白を恐れないでください。** 長い音と休みがリックをより大きく聞かせます。

- **音を詰め込みすぎる。** 余白がなければ余裕もありません。少なく弾いてもっと伸ばします。
- **拍を見失って本当に遅れる。** レイバックは「わざと」少し遅らせることです。心の中で拍は数え続けます。
- **すべての音を同じ強さで弾く。** 強弱がないと平坦です。大事な音は大きく、残りは小さく。
- **長い音をただ放っておく。** 伸ばしたルートにビブラートがないと死んだ音になります。余白にも命を吹き込みます。
