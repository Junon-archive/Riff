# Month 3 — 深化・統合：ブルースを話す
> カリキュラム: blues_3months · Month 3

今月を一文で: **ブルースを話す。** もうスケールをなぞるのではなく、コードに反応します。コードが変わればそのコードの音に**着地**し、マイナーとメジャーの3度を混ぜて**「BB」の色**を出し、**スライド・ベンド・ビブラート**を一つの文に編んでリックを成熟させます。最後は一人で<mark>12小節をコンピングとソロで行き来しながら</mark>一曲を完成・録音し、次の旅ソロ/スケールへ見送られます。

## 今月の旅
- **Week 9** — コード追いかけ。コードが変わればそのコードのルート・3度に着地。
- **Week 10** — メジャー/マイナー「BB」の色。b3→3の半音で成熟したブルースの色。
- **Week 11** — リック語彙の拡張 + フレージングの成熟。道具を編み、レイバックで余裕を。
- **Week 12** — リズム↔リードの対話 → 最終卒業・録音 + ソロ/スケール案内。

## 終わると変わること
コードが変わるたびに着地音を移して「チェンジを追う」音を出し、BBの色と成熟したフレージングでリックが「自分の声」に聞こえます。一人で12小節をコンピング（問い）とソロ（答え）で満たし、一曲のブルースを完成させます。この表現を指板全体・モードへ広げたくなったら、次の旅ソロ/スケールが待っています。
