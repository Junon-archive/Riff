# Day 1 — Aマイナーペンタトニックのボックス1（5フレット）を刻む

## ① 理論・説明

この4週間、私たちはリズムで「質問」する方法を身につけました。ここからは、その質問に**リードで答える**番です。答えを作るには、まずどの音を選べばいいか教えてくれる地図が必要で、その地図こそが**マイナーペンタトニックのボックス1**です。<mark>ボックス1はソロが思いきり遊べる遊び場です。</mark>Aブルースでは、この遊び場は**5フレット**の位置に広がります。ここに手を置けば、どの音を押さえてもブルースに合う響きが出るので、間違った音を心配する必要がありません。

ボックス1はたった五種類の音 — **R（ルート）・b3・4・5・b7** — が二オクターブにわたって繰り返される形です。手の形もとてもシンプルです。**人差し指（1番）は5フレット**を、**小指（4番）は8フレット**を担当し、ほとんどの音がこの狭いフレット幅の中で片づきます。<mark>一か所に手を置いてほとんど動かさなくても、六本の弦をすべて上り下りできます。</mark>だからこの形は、一日集中するだけでも手にすぐなじみます。

その中でも一番頼もしい目印は、**ルートA**の位置です。**6弦5フレット**、そして4弦7フレットと1弦5フレット — この三か所が、いつでも戻ってこられる家です。ソロで道に迷ったら、ルートに戻ればいいだけです。<mark>今日はフレーズを作りません。この形を手に刻むこと、その一つだけで十分です。</mark>最初は音がぶつ切りになっても大丈夫です。位置がなじめば、音は自然になめらかになります。まずは家になるルート三か所を目で確かめてみましょう。

## ③ 今日の練習（50分ルーティン）

**0〜10分・ウォームアップ**
BPM 70。急ぐ必要はありません。**ルートA三か所**を指で押さえながら手を目覚めさせます。<mark>三つのルートの位置が目を閉じても取れるか</mark>確かめます。

**10〜20分・頭のトレーニング（今日のターゲット＝ボックス1の位置を刻む）**
楽譜を見ずに、ボックス1の形を<mark>頭の中で一度描いて</mark>みます。どの弦の何フレットに点があるか声に出して言うと、もっと早く覚えられます。

**20〜40分・実戦：ボックス1を上り下り（BPM 70）**

**40〜50分・録音（今日のミッション）**
ボックス1を一度上り下りするのを録音します：<mark>十二の音が全部鳴っているか</mark>耳で聴いてみます。

**今日の完了基準：** Aマイナーペンタトニックのボックス1を低い弦から高い弦まで止まらずに上り下りし、ルート三か所を目を閉じても押さえられる。

## ④ ヒント / よくあるミス

ボックス1を覚えるときによく出るミスです。手の形さえきちんと取れれば半分は成功です。

```json
{
  "id": "m2.w5.d1.finger_tip",
  "type": "scale_shape",
  "meta": { "title": "Box 1 low string — 1st and 4th finger", "stringCount": 6, "tuning": ["E","A","D","G","B","E"], "key": "A minor" },
  "fretboard": { "startFret": 5, "fretSpan": 4, "dots": [
    { "string": 6, "fret": 5, "finger": 1, "label": "R", "isRoot": true, "role": "root" },
    { "string": 6, "fret": 8, "finger": 4, "label": "b3", "role": "scale" }
  ]}
}
```
▶ **人差し指は5、小指は8。** 6弦でこの指の配置を基本にすれば、ほかの弦も楽になります。

- **一本の指で全部押さえる。** 5フレットは人差し指、8フレットは小指 — 指を分担すればずっと速くなります。
- **音が濁って鳴る。** 指先を立ててフレットのすぐ後ろを押さえます。弦に軽く触れるだけでも音が濁ります。
- **一度に全部覚えようとする。** 今日はルート三か所と低い弦から。残りは繰り返せば自然に付いてきます。
- **速さから上げる。** 今日の目標は位置です。速さは今週のうちにゆっくり付けていきます。
