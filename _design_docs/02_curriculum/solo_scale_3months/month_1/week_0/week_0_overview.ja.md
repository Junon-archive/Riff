# Week 0 — オリエンテーション＆準備運動（中級者向け補正）
> カリキュラム：solo_scale_3months・Month 1・上位：../month_1_overview.md・dayKey: m1.w0

今週は新しい技術を覚えません。だからこそ、実は一番大事な週なんです。レンガを積み始める前に、「この家が最終的にどんな形になるのか」設計図を先に広げておくということですから。これから3か月がどこへ向かうのか、なぜ私たちが「3度に着地する」のか、その地図をまず目に入れて、毎日回すウォームアップルーティンとバッキング・録音環境を整え、「自分で音を選ぶ」という即興の感覚に初めて足のつま先だけ浸してみる週です。

## 今週のゴール
押さえるのは3つだけです。まず、このコースのエンジン（「コードが変わったら、そのコードの3度に着地する」）を頭で理解すること。次に、毎日繰り返すウォームアップルーティン（オルタネイトピッキング＋指の独立）を手にかけておくこと。そして、バッキングトラックをいくつかブックマークし、録音ツールをセットアップして「自分の音を聴く」環境を整えること。ここにAmペンタ第1ボックスでR（A）と♭3（C）の位置を目に刻んでおければ、Week 0は完璧に終わりです。

## なぜ今これをやるのか（対象について）
あなたは入門者ではなく、**「なんとなく中級者」**です――カバーはやったことがあるけれど、即興ソロは一度もやったことがない人。だから6弦の名前やフィンガリングといった基礎は思い切って飛ばします。もう知っていることをわざわざ握りしめておく理由はありませんよね。その代わり、本編のカリキュラム（3度インターバル→コードトーン・ターゲティング→モード→作曲）にすぐ乗れるように、必要なぶんだけ準備運動をして、自己診断で今の立ち位置を確認するのが今週の仕事です。短く、けれど正確に足場を作ります。

## 今週の道のり
- **Day 1** — コース全体の地図を広げます。「なぜよりによって3度なのか」（根音はただの名札で、明るさ・暗さを分けるのは3度）という考え方だけを頭に植え付けて、実際のトレーニングはまだしません。Amの3度がC（♭3）だと知っておくのが今日の全部。手は**1弦5フレットのA一音だけ**を使い、リズムで遊ぶ「一音即興」をやってみます。音が一つしかないぶん、かえってリズムや間が耳に入ってきます。
- **Day 2** — 毎日のウォームアップの原型を作ります。**スパイダーウォーク（クロマチック1-2-3-4）**でオルタネイトピッキングと指の独立を目覚めさせ（BPM 60でミスがなければ4ずつ上げる）、即興は一音から**二音（A＋C、♭3）**へと広げます。コードが鳴った瞬間にCで止まってみると、「あ、ぴたっとハマる」という3度着地の予告編を少しだけ味わえます。
- **Day 3** — Amペンタ第1ボックスの位置と各音の役割をリフレッシュします。上行・下行は手をほぐす準備運動として短く許すだけにとどめ、すぐに**3度インターバル（一音飛ばし）**を初めて本格的に味わいます。飛ばした音の響きに耳を開くのがポイントです。
- **Day 4** — メトロノームの規律（「弾けない速さでは練習しない」）を作り、YouTubeで「Am backing track」を検索して、遅いもの・ブルース風のものをいくつかブックマークします。録音ツールをセットアップし、自分の音を**タイミング・着地・間**の3つのチェックリストで聴く方法を身につけながら、この週を総まとめします。

## 核となる感覚（ターゲット音・考え方）
- **上行・下行はゴールではなく、あくまで準備運動です。** Week 0だけ手をほぐす目的で短く許容し、すぐに3度シーケンスへ移ります。
- **Amの3度＝C（♭3）。** まだ着地の練習はしません。「これから自分が踏む場所」として、その位置と響きだけを目と耳に刻んでおきます。
- **音の数をできるだけゼロに近く絞り込みます。** 一音→二音だけで遊びながら、華やかな指さばきよりリズム・ダイナミクス・間を先に聴く耳を作ります。上手い人ほど一音で人を泣かせるという言葉を、今週体感することになるはずです。

## 終わったら変わること
- 毎日のウォームアップルーティンが手に馴染んで、これから先どんな日でも5分あれば手が目覚めます。
- Amペンタ第1ボックスで、R（A）・♭3（C）の位置が目を閉じていても見えます。
- バッキングトラックをいくつかブックマークしていて、録音してタイミング・着地・間の3大チェックリストで自分の音をコーチできるようになっています。
- この4つがそろったら？ Month 1（本格的な即興）へ出発する準備は完了です。自己診断でXが多くても大丈夫です。このコースは、まさにそのXをOに変えるために作られているのですから。
