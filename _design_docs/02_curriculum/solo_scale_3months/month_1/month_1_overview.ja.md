# Month 1 — ペンタトニックでネックを制覇 → ブルースへ
> カリキュラム：solo_scale_3months・上位：../overview.md・範囲：Week 0〜4

ひとつだけお伝えすると、今までのあなたのソロは、「ペンタ第1ボックスの中を上行・下行で往復するだけ」がほぼ全部だったかもしれません。手はちゃんと動くのに、バッキングを流して「好きに弾いてみて」と言われた瞬間、ピタッと固まってしまう。今月は、その壁を壊す最初の月です。4週間が終わる頃には、指板上のAマイナーペンタトニックのどこに手があっても、コードが鳴った瞬間に「ここだ」と思える音に着地しながら、ブルースフレーズを紡げるようになっています。スケール運動が初めて「音楽」に変わる月です。

## 今月のゴール
一歩ずつ積み上げていきます。まずオリエンテーション（Week 0）でこのコースがどこへ向かうのか地図を広げ、毎日回すルーティンと環境を整えます。そのあとAmペンタトニックを**1ボックス→5ボックス、ネック全体**へと広げていきます。ここで大事なのは、ただ上下になぞるのではなく、上行・下行の惰性を手放して**3度インターバル・4連符（16分音符）シーケンス**で指板を隅々まで噛みしめること。その上に、このカリキュラムの心臓部である**3度（C、♭3）への着地**を植え付けます。最後の週にはブルーノート**♭5**と**ベンディング・ビブラート奏法**を乗せて、平坦だったスケールを歌うブルースフレーズに変えていきます。

## なぜ今これが大事なのか
即興の最初の一歩は、「ネックが広く見えること」と「終わりの音を意図的に選ぶこと」、この二つです。ひとつのボックスに閉じ込められているとアイデアはすぐ尽きてしまうし、終わりの音を適当に置くと、どんなに速く弾いても雑音のように聞こえてしまいます。Month 1はこの二つを同時に解決します。ネック全体を一つの地図としてつなげて手の行き場を広げ、「コードの3度に着地する」というエンジンを体に刻んで、どんなフレーズにもぴたっとハマる締めくくりを持たせます。このエンジンはMonth 2のコードトーン・ターゲティング、Month 3のモード・作曲までそのまま使い続ける骨組みなので、ここでしっかり刻んでおくと残りの2か月がぐっと楽になります。

## 週の流れ
| 週 | フォーカス | 終わったらできること |
|---|---|---|
| **Week 0** | オリエンテーション・自己診断・準備運動 | コースの考え方を理解し、毎日のウォームアップルーティンとバッキング・メトロノーム・録音環境をセットアップ、R・♭3の位置を頭に刻む |
| **Week 1** | ブロック1 + 3度・4連符シーケンス → 最初の着地 | ブロック1のシーケンスを両方向で使いこなし、バッキングの上でC/Aに意図的に着地してビブラートで締めくくる |
| **Week 2** | ブロック2・3を拡張 + スライドで連結 | ブロック1-2-3をスライドでつなぎ、ブロックを行き来しながら3度着地の即興を行う |
| **Week 3** | ブロック4・5 + ネック全体の視覚化 | 5ボックス全体をネック全体の地図として把握し、どこからでも一番近いCに着地する |
| **Week 4** | ブルース（♭5）+ ベンディング・ビブラート + ワンコードジャム | ♭5を経過音として使い、正確なベンディング・ビブラートを行い、ワンコードジャムを録音して自己診断する |

どの週も、前の週の上に積み上がっていきます。Week 0で位置とルーティンを固め、Week 1でひとつのボックスをシーケンスで完全に噛みしめたあと、Week 2・3ではその武器をそのまま持ってネック全体へと舞台を広げ、Week 4でその上にブルースの色と表現奏法を乗せる構造です。テンポは毎週少しずつ上がっていきますが（BPM 60前後から始めて80〜90まで）、ルールは4週間ずっと一つ——**弾けない速さでは練習しない。**

## 終わったらできること（月末の到達点）
- Amペンタトニックの5つのボックスをネック全体で一つの地図としてつなげ、3度インターバル・4連符シーケンスで両方向に貫通できる。
- ネックのどこにいても、フレーズの終わりを一番近いコードトーン（特に♭3＝C）に意図的に着地させられる。
- ブルーノート♭5を経過音として溶け込ませてブルースらしい色を出し、ベンディング（目標音程に正確に届かせる）とビブラート（手首）で着地音を歌わせられる。
- ワンコードのバッキングの上で固まらずに即興を続け、録音してタイミング・着地・間（余白）を基準に自分で自分をコーチできる。

## 来月（Month 2）へのつながり
Month 1が「マイナーペンタ＋♭3着地」という骨組みだとすれば、Month 2はそこに肉付けをしていきます。ペンタに2度・6度を加えて**ナチュラルマイナー**に拡張し（W5）、並行調のAm↔Cを行き来し（W6）、コードトーン1・3・5をマッピングし（W7）、そしてii-V-Iのガイドトーンラインへ（W8）と進んでいきます。Month 1で体に染み込ませた「3度着地」のエンジンが、そのままコードトーン・ターゲティングの土台になります。しっかり仕上げておけば、来月がぐっと軽く感じられるはずです。
