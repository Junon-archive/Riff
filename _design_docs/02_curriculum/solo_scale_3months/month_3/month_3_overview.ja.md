# Month 3 — モード・表現・作曲（卒業学期）
> カリキュラム：solo_scale_3months・上位：../overview.md・範囲：Week 9〜12

ここまで来られただけでも、本当にすごいことです。Month 1でペンタトニックのネックを掌握し、ブルースに触れ、Month 2ではダイアトニック・コードトーンターゲティング・ii-V-Iのガイドトーンまで手に馴染ませてきました。いよいよ最後の月です。この月の名前は「モード・表現・作曲」——言い換えれば**卒業学期**です。これまで積み上げてきた「正しい音を選ぶ力」の上に、初めて**色（モード）**を重ね、その色を**どう歌うか（表現）**を学び、最後にそのすべてを**あなただけの一曲（作曲）**に組み立てます。「指板の上で自分で音を選んで歌う」——このカリキュラムの最終目標を、まさにこの月で完成させます。

## この月に作るもの
Month 1・2で積み上げてきたペンタトニックのネック掌握・ブルース・ダイアトニック・コードトーンターゲティングの上に、**モードの色彩（ドリアン♮6、ミクソリディアン♭7）**を重ねます。ジャンルごとにモードをリアルタイムでスイッチングしながら、余白とフレージングでスケールの羅列を音楽に変えていきます。そして最後には、これまでのすべてを一つの**オリジナルソロとして作曲・録音し、自分でフィードバック**します。

## 4週間の流れをひと目で
| 週 | フォーカス | 終わったらできること |
|---|---|---|
| **W9** | ドリアンモード — 長6度♮6（F#）の色彩 | Am7・Am7–D7の上でF#を色彩として響かせ、コードごとの3度（C/F#）に着地した8小節即興を録音 |
| **W10** | ミクソリディアンモード — ♭7の緊張・長3度への解決＋ブルージーな響き | A7・A7–D7の上でブルーノートを長3度へ解決させ、2つのフォームをスライドでつなぎ、コードごとの3度（C#/F#）に着地 |
| **W11** | ジャンル別モードスイッチング＋余白・フレージングのミニレッスン | ブルース／ファンク／ロックのバッキングごとにモードを即座に選び、ジャンルらしいフレージングで3度を歌うように着地 |
| **W12** | オリジナルソロの作曲・レコーディング・セルフフィードバック | ドリアンとミクソリディアンが共存する8小節のオリジナルソロを作曲・録音し、4つの基準で自分でリテイク |

## この月の歩き方
Month 2で身につけたコードトーンターゲティングとii-V-Iのガイドトーンが、この月の着地エンジンになります。ここに初めて「モードの色」を重ねます——マイナー側の洗練された色（W9ドリアン）とドミナント側のブルージーな色（W10ミクソリディアン）で両翼を満たし、実戦のジャンルの中でリアルタイムに乗り換えながら（W11）表現まで重ねたのち、最後にオリジナルソロとして組み立てます（W12）。色を学び→歌い方を学び→自分で曲を作る、まさにこの順番です。

## この月が終わったら（月末／3か月間の最終到達点）
- マイナーバッキングにはドリアン（♮6）、ドミナント／ブルースにはミクソリディアン（♭7）＋ブルーノートを即座に選んで色をつけられます。
- ジャンル（ブルース・ファンク・ロック）が変わるたびに、モードとフレージングのムードをリアルタイムで切り替えられます。
- コードが変わるたびにそのコードの3度（ガイドトーン）に着地する感覚を、モードの色彩と組み合わせて使えます。
- 余白・モチーフ・コールアンドレスポンスで「練習」ではなく「音楽」としてフレージングできます。
- バッキングの上で自分だけの即興ソロを作曲・録音し、自分で採点・リテイクできます。

## そして、卒業
この3か月を始めた日を覚えていますか？バッキングを流しただけで手が固まってしまい、YouTubeのソロをコピーはしても、自分で音を選んだことは一度もなかった、あの場所です。W12最終日（Day 4）のセルフフィードバック・リテイクが、このカリキュラムの締めくくりです。コピーばかりしていた停滞期のプレイヤーが、自分でソロを作り、自分の演奏を耳で聴き分けてより良いテイクに録り直せるようになったら——それが卒業です。バラバラだったパーツが一曲にまとまるその瞬間、あなたはもう、始めた頃のあなたではありません。最後まで一緒に歩いていきましょう。
