# 3개월 솔로/스케일 마스터 커리큘럼 — Overview

> 커리큘럼 ID: `solo_scale_3months` · 상위: `../../00_master_prd.md`
> 구조: **Week 0(온보딩) + 3 Month × 4 Week**, 각 Week = **Day 1~4**. Day = 진도 원자 단위.

## 🎯 대상 레벨 (중요 — 이 코스는 입문자용이 아님)
**"애매한 중급자"** 가 정확한 타깃이다.
- 기타를 처음 잡는 사람이 **아니다.** 여러 곡을 쳐봤고 커버도 해봤다.
- 솔로는 **카피(따라치기)** 만 해봤고, **즉흥 솔로는 한 번도 시도해본 적 없는** 사람.
- 즉, 손은 어느 정도 돌아가지만 "지판 위에서 스스로 음을 골라 노래하는" 경험이 0인 정체기 플레이어.

따라서 6현 이름·핑거링 같은 **기초는 가르치지 않는다.** 대신 즉흥에 필요한 감각(스케일 시퀀싱 → 코드톤 타겟팅 → 모드 색채 → 작곡)을 빠르게 쌓아 올린다.
(선수 지식이 부족한 사람을 위한 최소 예열은 **Week 0**에서 짧게 보정한다.)

## 목표 (What you'll be able to do)
3개월 뒤: 5개 펜타토닉 박스를 넥 전체에서 연결하고, 코드가 바뀔 때마다 그 코드의 **3도(가이드톤)에 착지**하며, 도리안/믹솔리디안 색채를 얹어 **백킹 위에서 자기만의 즉흥 솔로를 구성·녹음**할 수 있다.

## 관통 철학 (모든 주차에 강제)
1. **상하행 단순 암기 금지 → 3도 인터벌 / 4연음 시퀀스로 접근.** 단, Week 0에서만 상하행을 *기계적 예열*로 짧게 허용(발판)한 뒤 곧바로 변형으로 넘어간다.
2. **항상 코드의 3도에 착지(코드톤 타겟팅).** 이 커리큘럼의 핵심 엔진.
3. **주 4일 × 50분 루틴** (워밍업 / 두뇌 훈련 / 실전 즉흥 / 녹음·자가 피드백).

## 주차별 요약
**Week 0 — 온보딩 & 예열 (중급자 보정)**
- d1: 코스 오리엔테이션 + "왜 3도 착지인가" 미리보기 + 자가진단 체크
- d2: 얼터네이트 피킹 & 핑거 독립 예열 루틴 (매일 워밍업의 원형)
- d3: Am 펜타 1박스 위치 리프레시 — 상하행으로 손 풀고 → 3도 인터벌 맛보기
- d4: 메트로놈 규율 + 백킹 트랙/녹음 앱 세팅 + 자기 연주 듣는 법

**Month 1 (W1~W4)** — W1 펜타 1박스(3도·4연음 시퀀스) / W2 2·3박스+슬라이드 연결 / W3 4·5박스+넥 전체 / **W4 블루스(♭5) + 밴딩·비브라토 주법 미니레슨 + 원코드 잼**
**Month 2 (W5~W8)** — W5 2·6도 추가→내추럴 마이너 / W6 나란한조 Am↔C / W7 코드톤(1·3·5) 매핑 / W8 ii-V-I 가이드톤(F→B→E)
**Month 3 (W9~W12)** — W9 도리안(♮6) / W10 믹솔리디안(♭7)+블루지 / **W11 장르별 모드 스위칭 + 여백·프레이징 미니레슨** / W12 오리지널 솔로 녹음·셀프 피드백

## 미니레슨 삽입 위치 (설계 결정)
- **밴딩/비브라토 주법 how-to → M1 W4.** 그동안 벤딩을 "사용"만 하고 안 가르쳤던 공백을 메움. 정확한 음정까지 밀어 올리는 벤딩 + 비브라토 컨트롤을 명시적으로 다룸.
- **여백/프레이징(쉼표·모티프·콜앤리스폰스) → M3 W11.** 타겟팅·모드가 갖춰진 시점에서 "스케일 나열"을 "음악"으로 바꾸는 단계. W12 작곡으로 이어짐.

## Day 문서 규약 (각 day_N.md 공통 템플릿)
1. **프론트매터** — 제목, dayKey(`m1.w1.d1`), 예상 소요(50분), 다국어 키.
2. **① 이론/설명** — 토스체. "오늘 왜 이걸 하냐면…"으로 시작.
3. **② 시각 자료** — 지판/타브 **JSON**. 반드시 `../../03_data_schema/fretboard_score_schema.json` **표준 스키마**를 따른다.
4. **③ 오늘의 연습(50분 루틴)** — 워밍업/두뇌/즉흥/녹음. 구체 BPM·타겟음·완료 기준 명시.
5. **④ 팁/흔한 실수**.

## 표준화 규약 (결정 반영)
- **녹음 도구 일반화:** 특정 앱(GarageBand 등) 의존 제거. "아무 녹음 앱 / 폰 음성 메모"로 표기(글로벌·크로스플랫폼). 매일 녹음 강제 대신 권장.
- **JSON 스키마 단일화:** 세션 A가 월별로 다르게 쓰던 필드(`interval`/`degree`/`highlight`/`target`/`tempo`/`tempo_bpm` 등)를 **전부 `fretboard_score_schema.json` 표준**으로 통일. 수합 시 `curriculum-architect`가 정규화하고 `notation-validator`가 검증한다.
